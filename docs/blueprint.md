# **App Name**: Asterasia Café

## Core Features:

- Hero Section: Visually striking hero section with café name, tagline, CTA for table reservations.
- About Section: Display café's story and highlight key aspects of Asterasia.
- Menu Highlights: Showcase selected food items (Coffee, Desserts, Specials) with enticing image placeholders and category labels.
- Booking Section: Offer options for pre-booking and party booking.
- Gallery Section: Display mosaic-style layout using abstract image placeholders. Users can ask a generative AI tool to generate a relevant background for the images.
- Contact Information: A 'chat with us' functionality which lets a user reach out to ask a question or clarify information

## Style Guidelines:

- Light Mode: Primary color warm orange (#E2852E), creating a warm, inviting atmosphere.
- Light Mode: Accent color soft golden yellow (#F5C857), providing subtle highlights.
- Light Mode: Background highlight light pastel yellow (#FFEE91).
- Dark Mode: Primary color deep muted teal (#36656B), adding a premium, sophisticated touch.
- Dark Mode: Accent color earthy green (#75B06F) for a natural feel.
- Dark Mode: Background highlight soft muted yellow (#DAD887).
- Headline font: Elegant serif with a modern café-style.
- Body font: Clean, readable sans-serif.
- Use irregular organic blobs, fluid curves, pebble-like shapes, and soft waves. Asymmetrical layouts with strong visual balance.
- Subtle, smooth transitions and loading animations to enhance user experience.
- Custom icons that fit the organic, flowing design, embedded within soft blob shapes.
